# wangyiyun-app

## Before project operation
```
1.需要node环境
2.还需要启动服务器
    将Fangwang-Music-Server项目的NeteaseCloudMusicApi-master文件夹使用运行起来
    加入文件夹，输入cmd进入命令行窗口
    输入node app.js
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
